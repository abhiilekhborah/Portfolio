# 🎮 Pixel Arcade Portfolio

A fully immersive, retro 8-bit/16-bit arcade game inspired portfolio website for Abhilekh Borah. 

This project is built with React, Vite, Tailwind CSS v4, Framer Motion, and GSAP, featuring a full pixel-art aesthetic, interactive RPG menus, CSS character sprites, and Web Audio API retro sound effects.

## 📦 Requirements & Installation

This project uses Node.js. All dependencies are already listed in the `package.json` file. 

To install the dependencies, you need to have [Node.js](https://nodejs.org/) installed on your computer.

1. Open your terminal in this project folder (`/Users/abhilekhborah/Desktop/Portfolio`).
2. Run the following command to install all required packages:
   ```bash
   npm install
   ```

## 🚀 How to Run the Project

Once the dependencies are installed, you can start the local development server by running:

```bash
npm run dev
```

After running the command, open your browser and navigate to the local link provided in your terminal (usually `http://localhost:5173/`).

## 🛠️ Tech Stack
- **React 19**
- **Vite 8**
- **Tailwind CSS v4** (using the new `@theme` CSS configuration)
- **Framer Motion** (for component animations)
- **GSAP** (for complex scroll triggers and HUD counting)

## 🏗️ How to Build for Production

When you are ready to deploy your portfolio to the web (e.g., Vercel, Netlify, or GitHub Pages), run:

```bash
npm run build
```
This will create a `dist/` directory with your optimized production files.
